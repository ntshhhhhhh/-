#NewsBot - это телеграм бот, который позволяет пользователю получать свежие новости. Бот использует NewsApi и удобные Inline-кнопки для навигации 
#Возможности - Получение случайной новости, Получение новостей по выбранной теме, Получение новостей по выбранному источнику и Поиск новостей по фразе/слову
